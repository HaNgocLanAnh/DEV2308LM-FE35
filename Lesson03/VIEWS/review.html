Tổng quan

        I, Table-bảng:
            1, cú pháp - cấu trúc: table > thead >tbody > tfoot 
                                            tr>th/td 
            2, các thuộc tính thường dùng: rowspan, colspan 
        II, Form - biểu mẫu:
            1, cú pháp - cấu trúc:
                form    + method
                        + action
            2, các thẻ nhập liệu:
                input: type(text, number, date, pasword,...)
                select :option 
                textarea 
                *, Lưu ý:   + với input thì mặc định type là text 
                            + cấc thuộc tính trong input, select, textarea 
                                    - name: quy định vùng lựa chọn 
                                    - id, class, attr-custom: xác định element 
                                    - value != placeholder và value != nội dung của option 
            3, submit, reset form
                element thường dùng: input/button với type tường ứng => page sẽ bị load lại 
                                     button với type="button"        =>page sẽ không bị load lại
